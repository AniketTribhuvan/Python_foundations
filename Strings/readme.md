# Strings in Python

## Overview

This folder focuses on string manipulation in Python.

Strings are one of the most important data types because they are heavily used in:
- automation
- web development
- APIs
- AI/NLP
- data processing

## Topics Covered

- String creation
- Indexing
- Slicing
- String methods
- Escape sequences
- String formatting
- Iteration through strings
- Immutability


## Skills Developed

- Text processing
- Pattern understanding
- Formatting outputs
- Working with immutable sequences


## Mini Projects / Practice

- text_analyzer.py
- Palindrome checker
- Word counter
- Vowel counter


## Key Takeaways

- Strings are immutable
- Indexing starts from 0
- Slicing creates new strings
- String methods return new objects
